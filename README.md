# Travel Bliss Website 🌍

## 📌 Project Overview
Travel Bliss is a responsive travel website created using HTML and CSS.

The website showcases:
- Popular travel destinations
- Travel services
- Hero section
- Responsive design
- Footer with social links and map

## 🛠 Technologies Used
- HTML5
- CSS3

## 🌟 Features
- Clean user interface
- Navigation menu
- Destination cards
- Service section
- Responsive layout
